Política de Privacidade

Última atualização: abril de 2026

Vigente a partir de 22 de abril de 2026

O Revisou respeita a privacidade dos usuários e realiza o tratamento de dados pessoais em conformidade com a Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018 – LGPD).

1. Dados Coletados

Podemos coletar as seguintes informações:

Dados fornecidos pelo usuário

  • Nome ou apelido
  • Endereço de e-mail
  • Credenciais de acesso

Dados de uso

  • Progresso de estudo
  • Interações com flashcards
  • Resultados de quizzes
  • Preferências de estudo

Dados técnicos

  • Endereço IP
  • Tipo de dispositivo
  • Sistema operacional
  • Versão do aplicativo
  • Registros de acesso

2. Finalidade do Uso dos Dados

Os dados coletados são utilizados para:

  • Permitir autenticação de usuários
  • Operar a plataforma
  • Registrar progresso de estudo
  • Melhorar funcionalidades
  • Garantir segurança da plataforma
  • Prevenir abusos ou fraudes

3. Compartilhamento de Dados

O Revisou não vende dados pessoais. Os dados poderão ser compartilhados apenas quando necessário para:

  • Operação da infraestrutura tecnológica
  • Cumprimento de obrigações legais
  • Atendimento a ordens judiciais

4. Armazenamento e Segurança

O Revisou utiliza medidas técnicas e organizacionais para proteger os dados dos usuários. Apesar disso, nenhum sistema é completamente imune a riscos.

5. Retenção de Dados

Os dados serão armazenados pelo tempo necessário para:

  • Funcionamento da plataforma
  • Manutenção da conta do usuário
  • Cumprimento de obrigações legais

6. Direitos do Usuário

Nos termos da LGPD, o usuário pode solicitar:

  • Acesso aos dados pessoais
  • Correção de dados
  • Exclusão de dados
  • Portabilidade de dados

7. Alterações nesta Política

Esta política poderá ser atualizada periodicamente. A versão mais recente estará sempre disponível na plataforma.
